Show Low Severity Threats

AI / ML Sensitivity
Anomaly Detection Threshold 0.65
Lower = more sensitive (more alerts). Higher = less sensitive (fewer false positives). Recommended: 0.55 – 0.75
SYN Flood Threshold (External IPs) 50
Number of SYN packets before an external IP is flagged as a SYN flood. Default: 50
Port Scan Threshold (External IPs) 20
Number of packets in 5 seconds before an external IP is flagged as a port scan. Default: 20
ML Training Sample Size 500
Packets collected before the Isolation Forest model trains. More = smarter model, but longer wait. Default: 500

Packet Filter
Which packets to capture from your network interface.
Save Normal Packets to DB
Store normal (non-threat) packets in the database. Disabling this saves disk space — only threats are stored.
Max Database Size
Maximum number of packet rows to keep. Older rows are automatically deleted.
